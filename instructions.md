ApplyTracker is an intelligent, full-stack job search lifecycle manager designed to transition the job hunt from a passive manual process to an active, AI-driven strategy. Built on a FastAPI/Uvicorn backend, the application integrates with real-time job board APIs (such as Adzuna or Jooble) to populate a "Discovery Engine." This engine utilizes together.ai (specifically the Llama 3 or Mixtral models) to perform semantic analysis, comparing a user's uploaded JSON-based resume against raw job postings to calculate a "Compatibility Score".

The application features a centralized Kanban-style dashboard that manages state transitions for each application—from initial discovery to the final offer. To solve the problem of generic outreach, ApplyTracker includes a Contextual Outreach Module that leverages the LLM to draft personalized follow-up emails and LinkedIn messages by synthesizing specific requirements from the job description with the user's unique project highlights.
